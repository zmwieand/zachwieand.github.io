Projects

What I work on in my free time




GeoTweet

  • Android application which can search Twitter based on query content and location.
  • Just enter what you would like to search and touch a position on the map view. The app will find 10 tweets matching the search query content that were within a 50 mile radius of the touched location.
  • Twitter4J and Android API
  • See it here


Thumb Track

  • Android Application which keeps track of how much a user scrolls on their phone in inches. Can also tell you the average velocity of every swipe in pixels per second.
  • Currently does not work as a service in the background, only in app.
  • Java (Android API)
  • Completed at ub.hacking 2014
  • See it here


Scrolr

  • Chrome Extension which allows Chrome to use the Users webcam to track their face and scroll up, down, left or right with their movements.
  • C++ with OpenCV API
  • Winner of Kent Hack Enough 2013


ACM Website
